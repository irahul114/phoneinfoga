# Digital Creator Services Web Application

A modern, premium digital creator services platform built with React, Tailwind CSS, and Framer Motion. This application provides professional services for creators including website design, social media content creation, account growth, digital marketing, video editing, and blogging tools.

## 🚀 Features

### Core Features
- **Dashboard** - Overview of services, active projects, and analytics
- **Services Section** - Comprehensive service offerings with pricing
- **AI Tools Integration** - AI-powered content generation tools
- **Portfolio** - Project showcase with filtering and search
- **Pricing Plans** - Flexible pricing with comparison tables
- **Client Dashboard** - Project tracking and communication
- **Admin Panel** - Complete management system
- **Contact & Support** - Live chat and support system
- **Authentication** - Multi-provider login system

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd digital-creator-services
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
